## Installation

Install the package in your project.

<CodeGroup>

  ```bash npm
  npm i @ng-primitives/avatar
  ```

  ```bash yarn
  yarn add @ng-primitives/avatar
  ```

</CodeGroup>

## Usage

Assemble the avatar directives in your template.

```html
<span ngpAvatar>
  <img ngpAvatarImage src="..." alt="..." />
  <span ngpAvatarFallback>NG</span>
</span>
```

## API Reference

The following directives are available to import from the `@ng-primitives/avatar` package:

### NgpAvatarDirective

There are no inputs or outputs for this directive.

### NgpAvatarImageDirective

There are no inputs or outputs for this directive.

### NgpAvatarFallbackDirective

<ResponseField name="ngpAvatarFallbackDelay" type="number">
  Define a delay before the fallback is shown. This is useful to only show the fallback for those with slower connections.
</ResponseField>

## Global Configuration

You can configure the default options for all avatars in your application by using the `provideNgpAvatarConfig` function in a providers array.

```ts
import { provideNgpAvatarConfig } from '@ng-primitives/avatar';

bootstrapApplication(AppComponent, {
    providers: [provideNgpAvatarConfig({ delay: 1000 })]
})
```

### NgpAvatarConfig

<ResponseField name="delay" type="number">
  Define a delay before the fallback is shown. This is useful to only show the fallback for those with slower connections.
</ResponseField>

## Schematics

You can use the `ng generate @ng-primitives/avatar:avatar` schematic to generate a new avatar component.

```bash
ng generate @ng-primitives/avatar:avatar
```

<ResponseField name="project" type="string">
